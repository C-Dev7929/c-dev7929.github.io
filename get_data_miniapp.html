🚀 Lấy Data Telegram Mini App

Công cụ trích xuất init_data từ Telegram Mini App

📋 Hướng dẫn sử dụng

1

Mở Telegram Web

Truy cập web.telegram.org trên trình duyệt (Chrome, Edge, Firefox...)

2

Mở Mini App

Tìm và mở Mini App (bot) mà bạn muốn lấy data. Ví dụ: Game bot, Trading bot...

3

Mở Developer Tools

Nhấn F12 hoặc Ctrl + Shift + I (Windows) / Cmd + Option + I (Mac)

4

Chuyển sang tab Console

Trong Developer Tools, click vào tab Console

5

Copy và chạy code bên dưới

Click nút "📋 Copy Code" bên dưới, paste vào Console và nhấn Enter

⚠️ Lưu ý quan trọng

• Chỉ chạy code này trên web.telegram.org khi Mini App đang mở
• Không share init_data với người khác (dữ liệu cá nhân)
• Code hoàn toàn chạy trên máy bạn, không gửi data đi đâu

💻 Code để chạy

// ========================================
// CODE LẤY DATA TỪ TELEGRAM MINI APP
// Tác giả: Tối ưu hóa và có comment
// ========================================

(async function () {
    try {
        console.log("🔎 Đang tìm Telegram Mini App...");

        // Tìm đúng iframe Telegram (lọc theo domain)
        const iframe = [...document.querySelectorAll("iframe")]
            .find(f => f.src && f.src.includes("tgWebAppData"));

        if (!iframe) {
            throw new Error("Không tìm thấy iframe chứa tgWebAppData");
        }

        console.log("✅ Đã tìm thấy iframe Mini App");

        // Lấy hash từ src (Telegram luôn nhét data ở đây)
        let hash = iframe.src.split("#")[1];

        if (!hash) {
            throw new Error("Không tìm thấy hash chứa tgWebAppData");
        }

        const hashParams = new URLSearchParams(hash);
        const tgWebAppDataEncoded = hashParams.get("tgWebAppData");

        if (!tgWebAppDataEncoded) {
            throw new Error("Không có tgWebAppData trong URL");
        }

        // Decode nhiều lớp (Telegram encode 2 lần)
        let initData;
        try {
            initData = decodeURIComponent(tgWebAppDataEncoded);
        } catch {
            initData = tgWebAppDataEncoded;
        }

        // Parse init_data querystring
        const parsedInitData = Object.fromEntries(
            new URLSearchParams(initData)
        );

        // Parse user JSON (field này lại encode thêm lần nữa)
        let user = {};
        if (parsedInitData.user) {
            try {
                user = JSON.parse(decodeURIComponent(parsedInitData.user));
            } catch {
                user = JSON.parse(parsedInitData.user);
            }
        }

        const result = {
            init_data: initData,              // gửi server để verify
            query_id: parsedInitData.query_id || null,
            auth_date: parsedInitData.auth_date || null,
            hash: parsedInitData.hash || null,

            user: user,
            full_name: [user.first_name, user.last_name]
                .filter(Boolean)
                .join(" ") || null,

            avatar: user.photo_url || null,
            id: user.id || null,
            username: user.username || null
        };

        console.log("✅ ====== LẤY DATA THÀNH CÔNG ======");
        console.log(result);

        // Copy clipboard (API mới, không bị browser chặn)
        await navigator.clipboard.writeText(
            JSON.stringify(result, null, 2)
        );

        console.log("📋 Đã copy JSON vào clipboard!");

        return result;

    } catch (err) {
        console.error("❌ Lỗi:", err.message);
        console.log("💡 Hãy đảm bảo:");
        console.log("   1. Đang mở https://web.telegram.org");
        console.log("   2. Mini App đã load xong");
        console.log("   3. Chạy lại script sau khi app hiện hoàn toàn");
    }
})();
✅ Kết quả khi chạy thành công

Bạn sẽ thấy trong Console:
• ✅ LẤY DATA THÀNH CÔNG!
• Object chứa: init_data, full_name, avatar
• 📋 JSON đã được tự động copy vào clipboard!
• Bạn có thể paste (Ctrl+V) để sử dụng

ℹ️ Giải thích kết quả

• init_data: Dữ liệu xác thực để login vào Mini App
• full_name: Tên đầy đủ của bạn trên Telegram
• avatar: Link ảnh đại diện (nếu có)

❓ Các lỗi thường gặp

❌ Không tìm thấy Mini App hoặc tgWebAppData

Nguyên nhân: Bạn chưa mở Mini App hoặc đang ở sai trang
Giải pháp: Đảm bảo đang ở web.telegram.org và đã mở Mini App

❌ Không có tgWebAppData

Nguyên nhân: Mini App chưa load xong hoặc không hợp lệ
Giải pháp: Đợi Mini App load xong rồi chạy lại code

❌ Lỗi khi parse JSON

Nguyên nhân: Dữ liệu không đúng định dạng
Giải pháp: Refresh trang và thử lại
